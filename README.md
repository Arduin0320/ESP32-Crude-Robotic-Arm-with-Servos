# ESP32-Crude-Robotic-Arm-with-Servos
An introduction into robotics using simple electrical components. 

Bill of Materials (BOM) for ESP32 Robotic Arm
1x ESP32 Development Board: Standard version (e.g., ESP32-WROOM-32).

4x MG90S Servomotors: Micro-servos with metal gears for mechanical durability at the arm joints.

4x Rotary Potentiometers: Linear taper, recommended value of 10k ohms.

1x 5V DC Power Supply: Minimum 3A capacity, though 5A is highly recommended. Note that the stall current for a single MG90S can reach approximately 700mA.

4x Ceramic Capacitors: 100nF. Used for decoupling the analog signal of the potentiometers, installed between the wiper and ground (GND).

1x Jumper Wires Set: Male-to-Male and Male-to-Female for prototyping connections.

1x or 2x Breadboards: For component assembly and power distribution lines.
